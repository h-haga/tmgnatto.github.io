僕の制作環境について

僕のweb制作環境はこれだ・・・。全く人にオススメできない時代遅れの開発環境を、恥を忍んでご紹介します。web制作ってこんなに簡単にできちゃうんだというのを1mmでも感じることができれば幸いです・・・。（あくまでも僕の環境なので）

はじめに

基本的に必要なものはhtmlとcss、あとは画像くらいです。アップすることは無視してデザインするということだけを考えてます。htmlとcssに関してはSublime Text 2というテキストエディタを使ってコーディングしています。後述しますがプラグインが豊富でいくつかの手間を省くことができる優れものです。が、時代遅れ感はあります。今の時代に乗っていそうなのはatom、Sublime Text 3、BracketsやVisualStudioCodeというやつかもしれませんし、そうでないかもしれません。画像に関してはphotoshopとillustratorを主に使っています。とりあえずこれがあれば全てのデザインができます。ただ僕はここにきて初めてSketchなるものを使うようになりました。用途に応じると良いやもしれませんね。

スキルや環境について

スキルとはhtmlのタグやcssのプロパティを覚えることです。実際に自分でコーディングして覚えるもよし、既にあるサイトを参考にするもよし、様々な手段で引き出しを増やして行くと良いと思います。

そうしてある程度こなれてくると、人間ですから楽をしたいと思うようになります。自分にあった環境制作を作るようになるのです。もっと簡単に打つことはできないものかと。そこでSublime Text 2です。Sublime Text 2にはプラグインというものがあり、エディタに新しい機能を追加することができます。ここでは簡単にこれらを入れておくといいのかもしれないというのをピックアップして見ました。

僕のプラグイン

  • CSS Snipets

    CSSの自動補完をしてくれます。

  • Emmet

    ctrl+Eすれば色々展開してくれて、楽ができます。

  • HTML5

    HTML5の自動補完をしてくれます。

  • Sass

    Sassの自動補完をしてくれます。

  • SublimeLinter

    リアルタイムで文法のチェックをしてくれます。閉じ忘れがなくなります。

これ以外にも本当に様々なものがあります。自動整列やらファイル名を補完したりなど。今回は僕が使っていなかったためここでは紹介はしませんでした。googleに聞くと素敵な回答がたくさん得られるかと思います。
